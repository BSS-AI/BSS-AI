        '''Crafting''' is a [[Mechanics|mechanic]] that consists of using the [[Blender]] in the
        [[Badge Bearer's Guild]] to turn crafting ingredients into other [[items]].

        == Usage ==
        The first menu will offer 20 options for items to be crafted, as seen below. The options can
        be navigated by clicking the left or right arrows. Select the item to be crafted by clicking
        the green button saying \"Make (item name)\".

        The next menu allows the player to set any number of that item to be produced, up to the max
        number that can be allowed with the player's ingredients. Pressing \"Confirm\" starts the
        crafting process. Pressing \"Cancel\" takes them back to the first menu.
        [[File:RedExtract-0.png|thumb|215x215px]]
        When an amount is confirmed, the interface is replaced with a screen showing the item being
        produced, the number of said crafting material made so far and to be made, and the time left
        until all of the items are made. There is a button to \"End Crafting,\" and a button to \"Speed
        Up for # [[Ticket]]s\". If the player clicks \"End Crafting\", the blender will refund any
        materials that have not completed crafting. The amount of tickets required to speed up the
        ingredient varies. For every 5 minutes left, the Blender requires 1 more ticket to speed up.
        [[Moon Charm|Moon charms]] and [[gumdrops]] only require 1 ticket for every 50 minutes on
        the timer.

        It takes 5 minutes to craft each item and ending crafting at any time adds the items
        produced and the ingredients not yet used into the inventory. It also counts credit toward
        [[quests]] that require the player to use the Blender for the number of items produced. The
        time remaining for the items to be crafted only decreases while in-game. If the player
        leaves the game either by losing connection or exiting, the timer will pause until they join
        the game again.

        {{clear}}
        == Crafting Recipes ==
