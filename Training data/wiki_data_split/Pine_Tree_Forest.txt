        of single, double, and triple.

        This field can spawn [[honey]], [[Blueberry|blueberry]], [[ticket]], [[Blue Extract|blue
        extract]], [[Magic Bean|magic bean]], and [[Neonberry|neonberry]] tokens while collecting
        [[pollen]]. [[Planter]]s in this field grant extra [[whirligig]]s and blue extracts.

        [[Bees]] that like the Pine Tree Forest are [[Bear Bee]], [[Bumble Bee]], [[Cool Bee]],
        [[Bucko Bee]], [[Bubble Bee]], [[Carpenter Bee]], [[Cobalt Bee]], [[Festive Bee]], [[Fuzzy
        Bee]], and [[Tadpole Bee]]. Bees that dislike the Pine Tree Forest are [[Rascal Bee]], [[Rad
        Bee]], [[Riley Bee]], [[Fire Bee]], [[Shy Bee]], [[Precise Bee]], [[Digital Bee]] and [[Baby
        Bee]].

        This field gives Comforting [[Nectar]] when a planter is placed here.

        == Obtaining Boosts==
        [[File:Fis_pinetree.png|thumb|75px|The Pine Tree Forest boost icon.]]

        *Using the [[Blue Field Booster]] may give a Pine Tree [[Field Boost|Forest Boost]]. When it
        does, it will give +100% Pine Tree Forest pollen for 15 minutes.
        *Using [[glitter]] will add 100% Pine Tree Forest pollen onto the current boost for 15
        minutes if the current boost isn't already maxed out.
        *Using [[Field Dice|field dice]], [[Smooth Dice|smooth dice]], or [[Loaded Dice|loaded
        dice]] may add +100% - +300% Pine Tree Forest pollen for 15 minutes if the current boost
        isn't already maxed out.
        *Pine Tree [[Field Wind|Forest Winds]] can be obtained through the [[Wind Shrine]] or
        [[codes]]. They have a base of 20% instant Pine Tree Forest conversion and give +5% instant
        Pine Tree Forest conversion and +15% Pine Tree Forest pollen for each stack. Pine Tree
        Forest Winds can stack up to a total of 13 times.
        * The [[Pinecone]] [[Beequip]] can grant up to +15% Pine Tree [[Field Capacity|Forest
        Capacity]] and +10% Pine Tree Forest Pollen.
        * Redeeming certain valid codes.
        **WordFactory (Activates Pine Tree Forest Code Buff + other stuff).
        **PlushFriday (Gives Pine Tree Forest Code Buff + other stuff).
        **DemiDecade (Gives Pine Tree Forest x4 + Pine Tree Winds x15 + other stuff).

        ==Trivia ==
        *Festive Bee likes the Pine Tree Forest, which may be a reference to the fact that most
        Christmas trees are pine trees.
        **It is also the only red bee to like a field of the opposite color. The only other bee that
        shares this trait is [[Vicious Bee]] liking the [[Rose Field]].
        *The Pine Tree Forest is the only field with \"Forest\" in its name instead of \"Field\" or
        \"Patch\".
        *This field is guarded by the werewolf and 2 mantises. This makes the Pine Tree Forest the
        field with the most [[mobs]] guarding it, at 3.
        *There is a [[Royal Jelly|royal jelly]] token on top of the tallest pine tree in this field.
        To get it, the player can do the [[Obstacle Courses#Cloud Obby|Cloud Obby]] starting near
        [[Polar Bear]] using the [[Parachute]] to glide to the next step. However, it is much easier
        to use the [[Red Cannon]] and the [[Parachute]]/[[Glider]].
        *As of the [[Updates#2020.06.06|6/6/2020 update]], [[Honey Bee (NPC)]] can be found on the
        gray platform behind the Pine Tree Forest.
        *The Pine Tree Forest has shrunk since it was released. Before the Badge Bearer's Guild was
        added, the field stretched all the way to where the Badge Bearer's Guild currently is. When
        the Badge Bearer's Guild was added, the field was shrunken to end at the door of the guild.
        It has been shrunken again, now being farther from the guild's door.
        **One of the pine trees still remains near the [[Honey Bee Gate]] from the field's original
        spot.
        *This field currently has the most amount of bees disliking it, at 8.
