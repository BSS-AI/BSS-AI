        '''Memory Match''' is a match-style minigame located throughout the map. The goal is to
        match as many pairs possible in order to win [[items]]. There are four (five during Beesmas)
        of these venues that can be used. Each venue costs a varying amount of [[honey]] (See
        \"[[Memory Match#Locations|Locations]]\").

        == Gameplay ==
        [[File:Ossas.png|thumb|220x220px|The Extreme Memory Match screen.]] When activated, a screen
        will pop up with a 4x4 (5x4 for Extreme and Winter), card screen. A player has a limited
        amount of chances to match cards together. Matching a pair also costs a chance. They can
        gain more chances if they have the required [[Badges|badge]] and its rank in order to get
        it. Venues with rarer rewards require more honey, a better badge rank, and a longer
        cooldown.

        If at least one match is made, a message will appear in the chat, saying, \"Winner!\", with
        the items received shown afterwards.

        If no matches are made, a message will appear in the chat, saying, \"Better luck next time!\"

        === Winter Memory Match ===
        The '''Winter Memory Match''' is a limited-time memory match added in the
        [[Updates#2020.12.25|Beesmas 2020 update]], and reintroduced during Beesmas 2021 along with
        Part 2 of Beesmas 2022. It is located near the [[Pine Tree Forest]] and the [[Gift Boxes|Tan
        Gift Box]]. It costs 100 [[snowflake]]s to use with a base amount of five chances. Its
        cooldown is 4 hours. The player could only play it after completing that year's Spirit Bear
        Beesmas quest.
        When a player attempts to use this without completing Spirit Bear's Beesmas quest, the
        following message appears: \"''Looks like you need a membership of some sort to play winter
        memory match...''\".

        == Trivia ==
        * The Extreme Memory Match is the biggest Memory Match in terms of size of the \"computer\".
        It is also the most expensive Memory Match in terms of honey.
        * If the player starts a new memory match game while one is already active, the new one will
        not start, and the fee and new cooldown for the next memory match will still happen.
        ** Note that this is no longer possible, because the players movespeed is now set to 0 while
        in a memory match game, meaning it's impossible for them to move anywhere.
        * The Winter Memory Match is the only match that required completion of a quest, being
        [[Spirit Bear|Spirit Bear's]] for the [[Wind Shrine#Galentine Shrine|Galentine Shrine]].
        * A bug in Beesmas 2021 caused the prompt for Winter Memory Match to be broken. When the
        player did not have enough snowflakes, it would display: \"and Complete 1 Quest to You need
        100 Snowflakes to play Winter Memory Match\".
        [[File:Memory match (Winter).webp|thumb|The prompt for Winter Memory Match when the player
        didn't have enough snowflakes.]]
        * The chances of getting any prizes on every Memory Match excluding Extreme Memory Match are
        about 37% for 3 chances, 46% for 4 chances, 54% for 5 chances, and 60% for 6 chances. The
        Extreme Memory Match has a 45% chance to reward the player if they receive 5 chances, 51%
        for 6 chances, 56% for 7 chances, and 61% for 8 chances.
        * Winter Memory Match almost always has 2 pairs of Gingerbread Bears.
